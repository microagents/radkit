Radkit is a Rust library designed to help you build powerful, robust, and [A2A-compliant](https://a2a-protocol.org) agents with an elegant and productive developer experience. It prioritizes developer control, allowing you to manage agent behavior, execution flow, and state with precision.

## Core Features

<CardGrid>
  <Card title="A2A-Native" icon="approve-check">
    First-class support for the Agent-to-Agent communication standard, built-in from the ground up.
  </Card>
  <Card title="Powerful Tooling" icon="setting">
    Go beyond simple chat with structured outputs, multi-turn tool execution, and stateful tools.
  </Card>
  <Card title="Unified LLM Interface" icon="puzzle">
    A single, consistent API for major LLM providers like Anthropic, OpenAI, Gemini, and more.
  </Card>
  <Card title="Rust-Powered Reliability" icon="rocket">
    Leverage Rust's type safety and performance to build correct and efficient agent systems.
  </Card>
  <Card title="WASM-First Deployment" icon="laptop">
    Deploy agents as lightweight, secure, and portable WebAssembly modules.
  </Card>
</CardGrid>

## Who is this for?

- **Rust Developers** who want to build sophisticated AI agents without sacrificing control or performance.
- **AI Engineers** looking for a robust framework to create reliable, production-ready agent systems.
- **Innovators** who want to build applications on the open, interoperable A2A agent communication standard.
